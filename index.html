1
Acceso al sistema SAP desde la RF
P 01
¿Cuál es el primer paso para iniciar el proceso de traslado desde la terminal RF?
El primer paso es seleccionar Internet Explorer para acceder a SAP e ingresar los datos de usuario y clave de acceso proporcionados por el jefe inmediato.
P 02
Dentro del menú principal, ¿qué opción debe seleccionar el operario para iniciar el traslado de una UA?
Se debe seleccionar la opción 3. Traslado y luego elegir 1. Traslado UA para iniciar el proceso de traslado de la unidad de almacenamiento.
2
Escaneo de UA y selección de ubicación destino
P 03
Al aparecer la pantalla de traslado, ¿qué debe hacer el operario primero y qué tecla presiona para continuar?
El operario debe escanear la UA que va a trasladar y presionar F4. Cont para continuar al siguiente paso del proceso.
P 04
Al almacenar en un nuevo anaquel, ¿qué pasos debe seguir el operario además de escanear el nuevo código de verificación?
Al almacenar en un nuevo anaquel se debe cambiar el código de verificación, presionar F7.NvUb (nueva ubicación), escanear el nuevo código y confirmar con F1.Grab.
3
Confirmación y mensajes del sistema
P 05
¿Qué indica el mensaje "Se ha creado la orden de transporte" al finalizar el proceso?
El mensaje de orden de transporte creada indica que el proceso concluyó de forma exitosa y la UA fue trasladada y ubicada correctamente. Se presiona F3 o selecciona OK para continuar.
P 06
Según las instrucciones generales, ¿qué debe acompañar siempre a todo movimiento físico de mercancía?
Todo movimiento físico debe ir acompañado de su correspondiente movimiento en el sistema SAP. Cada acción real debe tener su registro lógico.
