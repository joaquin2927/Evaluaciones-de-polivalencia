1
Inicio del proceso y ruta en el sistema
P 01
¿Cuál es la ruta correcta en el menú SAP para iniciar el proceso de palletizado UA Caja?
Para iniciar el palletizado se debe ir a Proceso EM (Entrada de mercancía) y seleccionar la opción Embalar Cajas, que convierte las UA caja a pallet para almacenar en MAIN.
P 02
Al escanear cada caja individual, ¿qué debe verificar el operario en pantalla y qué tecla usa para continuar con la siguiente caja?
Cada vez que se escanea una caja se debe verificar que la información cargue en la parte inferior y que el contador UA Caja Total aumente. Se presiona Enter o F1 Grabar para continuar con la siguiente caja.
2
Cierre del proceso y mensajes de error
P 03
Al terminar de escanear todas las cajas, ¿qué debe verificar antes de cerrar y cómo procede?
Antes de cerrar se debe confirmar que el total de cajas en pantalla coincida con el conteo físico. Una vez verificado se presiona F4 Cerrar para finalizar el proceso.
P 04
¿Qué indica el mensaje "UA-Pallet ya existe" y qué debe hacer el operario?
El mensaje "UA-Pallet ya existe" indica que el UA Pallet que se está usando ya está armado o está repetido. Se debe seleccionar F3 Atrás para regresar y verificar el número de pallet.
3
Confirmación final del proceso
P 05
¿Para qué cajas aplica el proceso de convertir de UA caja a pallet?
El proceso aplica para las cajas que se encuentren en el almacén de tránsito 801 – Z03-Z04-Z05 provenientes de entradas o devolución de PCK.
P 06
¿Qué condición debe cumplir el UA Pallet antes de iniciar el proceso de escaneo?
El UA Pallet asignado debe estar previamente impreso antes de iniciar el proceso de escaneo y palletizado.
